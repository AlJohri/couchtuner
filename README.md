### Setup

```
npm install -g phantomjs-prebuilt
pipenv install
```

### Usage

```
pipenv run python couchtuner.py <url>
```

- **protip 1**: use vlc to watch the video as its downloading (a la streaming)
- **protip 2**: vlc has a command line tool
